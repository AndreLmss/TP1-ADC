# PeriféricosTech

Aplicação de gestão de venda e suporte de periféricos para PC.

## Descrição
Sistema desenvolvido em Python que funciona na consola de texto
para gerir clientes, produtos, vendas e suporte técnico de uma
loja de periféricos para PC.

## Funcionalidades
- Gestão de clientes (CRUD)
- Gestão de produtos (CRUD)
- Registo de vendas
- Suporte técnico via tickets
- Sistema de login com papéis
